# Setup 

## Docker
If you have Docker installed in your machine you can simply run:
```bash
$ docker-compose up
```
## locally:
```bash
$ cargo +nightly run
```
Or with hot-loading:
```bash
$ cargo watch -x run
```

# testing

Run tests:
```bash
$ cargo test
```

# Additionally
Rust format:
```bash
$ cargo fmt
```

Rust lint:
```bash
$ cargo clippy
```
