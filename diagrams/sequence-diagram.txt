# source code for sequence diagram on https://sequencediagram.org/

==Once per Pogram==
Verifier<->Prover:setup ceremony
Prover->Zokrates API:compile .zok program
Prover<-Zokrates API:return <proof-id>

Prover->Zokrates API:POST /<proof-id>/proving-key
Prover<-Zokrates API:return ACK

==Once per Proof==
Prover->Zokrates API:POST /<proof-id>/compute-generate-proof
Prover<-Zokrates API:return proof.json
Verifier<-Prover:verify proof.json
